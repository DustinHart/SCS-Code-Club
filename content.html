Welcome!

The SCS Code Club is a student-focused club for students who want to learn about web development. We meet on Monday afternoons until 5pm in room 233 (Mr. Hart's Room). The SCS Code Club is open to any SCS students who are excited about learning to code. Students will need their laptops for code club, so students should hold on to their laptops and bring them to code club. They will be stored in Mr. Hart's room until the next morning. They can pick it up on Tuesday morning, in Mr. Hart's room.

Students will begin by learning HTML basics and how a webpage works. Then students will explore other HTML, CSS, and JavaScript topics that they are interested in learning. The Code Club is focused on students, so if students have a particular ideas and topic they want to explore, we will spend time working on it.

If you have any questions, please email me at Dustin.Hart@dpsnc.net.

The Basics of CSS

The Basics of CSS

This tutorial is focused on the fundamentals of Cascading Style Sheets (CSS). While this tutorial is focused on external style sheets, beginners often start with changing the style of a webpage in the head of an html file. You can also change the style of elements in a page by writing the code within individual element tags. This is fine for beginners, but it is recommended that as you learn more about html and css that you move the CSS to an external style sheet. This allows web developers to easily change style of multiple webpages that are linked to the same style sheet.
